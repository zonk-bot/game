# game

一个简单的网页火柴人格斗游戏，入口文件为 `index.html`。

## 网页视频字幕识别助手

`scripts/web-video-subtitle-assistant.user.js` 是一个可安装到浏览器用户脚本管理器（如 Tampermonkey / Violentmonkey）的脚本，用于在当前网页中自动收集视频字幕。

### 功能

- 自动扫描当前页面的 `<video>` 字幕轨道（`TextTrack` / `VTTCue`）。
- 兼容常见播放器的可见字幕 DOM（例如 YouTube、Video.js、JW Player 等字幕容器）。
- 只保留包含中文或英文的字幕文本。
- 在页面右下角显示控制面板，可开始/暂停识别、清空结果。
- 支持导出：
  - `*-subtitles.txt`：按时间轴整理后的字幕文本。
  - `*-subtitles.json`：包含来源、语言、开始/结束时间、字幕内容的结构化数据。
  - `*-summary.txt`：基于词频的抽取式摘要与字幕统计信息。

### 使用方式

1. 安装 Tampermonkey 或 Violentmonkey。
2. 新建用户脚本，并复制 `scripts/web-video-subtitle-assistant.user.js` 的全部内容。
3. 打开带有视频字幕的网页，脚本会自动启动。
4. 播放视频后，右下角面板会显示已收集的中文/英文字幕数量。
5. 点击“导出字幕”“导出 JSON”或“导出摘要”保存文件。

### 注意事项

- 脚本优先读取网页已经提供的字幕轨道或页面上可见的字幕文本；它不会绕过浏览器安全限制去录制系统声音。
- 如果某个网站把字幕绘制在视频画面中（硬字幕）且没有提供字幕 DOM 或文本轨道，浏览器脚本无法直接读取画面文字，需要额外接入 OCR 服务。
- 摘要为本地抽取式总结，不会上传字幕内容；如果需要更强的生成式总结，可以把导出的字幕文件再交给大语言模型处理。
